
## Example
```
git clone git@github.com:olivianate/dev-server.git
npm install
npm run start 
```

### config src/abc.json 

---

| 参数名            | 默认值  | 说明     |
| -------------- | ---- | ------ |
| port | 3000 | 本地服务端口      |
| open |      | 使用默认浏览器访问页面 |
| root |  | 本地文件      |
| proxy |      | 跨域请求 |
