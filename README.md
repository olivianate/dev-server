# dev-server

实现页面自动刷新，跨域请求

## Example
```
git clone git@github.com:olivianate/dev-server.git
npm install
npm run start 
```

### 配置 src/abc.json 文件

---

| 参数名            | 默认值  | 说明     |
| -------------- | ---- | ------ |
| port | 3000 | 本地服务端口      |
| open |      | 使用默认浏览器访问页面 |
| root |  | 本地文件      |
| proxy |      | 跨域请求 |
